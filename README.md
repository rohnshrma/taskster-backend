# Backend (MERN Todo API)

## Setup

1. Copy environment file:

```bash
cp .env.example .env
```

2. Install dependencies:

```bash
npm install
```

3. Run server:

```bash
npm run dev
```

Server runs on `http://localhost:5001` by default.

## Endpoints

- `GET /api/health`
- `GET /api/todos`
- `POST /api/todos`
- `PUT /api/todos/:id`
- `DELETE /api/todos/:id`
